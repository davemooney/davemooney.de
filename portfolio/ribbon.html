Ribbon

Max for Live Ableton Live Control Device

Ribbon is a M4L device that allows gestural control of parameters in Ableton Live (via the Live API) using the Mad Catz Gametrak controller. It was used as part of the Le Doux Collective performance at Fusion Festival in 2014.
